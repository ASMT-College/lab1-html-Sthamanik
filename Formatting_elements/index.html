Text Styling
<b> Draws attention to text without conveying extra importance Bold text
<strong> Indicates strong importance, seriousness, or urgency Strongly emphasized text
<i> Represents text in an alternate voice or mood Italic text
<em> Emphasizes text with stress emphasis Emphasized text
<mark> Highlights text for reference purposes Highlighted text
<small> Represents side-comments and small print Smaller text
<u> Indicates text that should be stylistically different, like misspelled words Underlined text
<s> Represents text that is no longer accurate or relevant Strikethrough text
<Teletype> Represents text in a typewriter style Monotype text
<sub> and <sup> Represents subscript and superscript text respectively H2O and E=mc2
Text Direction and Language
<bdo> Overrides the current text direction Right-to-left text
<bdi> Isolates text that might be formatted in a different direction Bi-directional text
<dfn> Represents the defining instance of a term The HTML is a markup language
